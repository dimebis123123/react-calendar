<h1>💫Календарь событий с гостями и личным кабинетом</h1>
<h6>♥️Демо-ролик проекта можно посмотреть в самом низу!♥️</h6>
<p>
	Реализовал проект где вы можете зарегистрироваться и войти в календарь где создаются события, имеется возможность добавлять других пользователей в придуманные события.
</p>
<h2>🔦 Технологии с которыми работал</h2>
<ul>
	<li>React.js</li>
  <li>Typescript</li>
	<li>Redux toolkit</li>
  <li>Ant design библиотека ui</li>
	<li>СSS</li>
	<li>Node.js</li>
	<li>Express</li>
	<li>ORM Sequelize</li>
  <li>REST API AXIOS</li>
  <li>JWT токен</li>
  

</ul>
<h2>🦊 Основные функции что выполнимы в проекте </h2>
<ul>
    <li>Регистрация для создания событий</li>
    <li>Создание событий с упоминанием пользователей так-же зарегистрированных в приложении </li>
    <li>События подгружаются при загрузке</li>
    <li>Пользователь так-же не теряет сессию при обновлении страницы благодаря токену и локал хранилищу</li>
   
</ul>
<h2>🔆 Что я углубил в рамках разработки</h2>
<ul>
    <li>Первая полноценная работа с Typescript и соответственно углубление в него)</li>
    <li>В рамках приложения работали с Redux Toolkit, куда сложнее чем Mobx. Store, slice, reducers, dispatch, selector, много всего но с другой стороны всё структурированно</li>
    <li>Укрепил немного серверную часть так как по прошлой практике реализовывал под это приложение</li>

</ul>
<h2>♥️ Что можно улучшить</h2>
<ul>
    <li>По календарю можно выделять много разных моментов от создателя до в целом анимаций</li>
    <li>Реализовать логический функционал , например , не давать создавать события в прошлом</li>
    <li>Ошибки ошибки ошибки, есть вайб что можно куда лучше и больше отслеживать</li>
    <li>Добавить больше анимаций и интерактива</li>
  
</ul>
<h2>🪝 Как локально запустить этот проект у себя</h2>
<ul>
    <li>1. Клонировать репозиторий</li>
    <li>2. Установить зависимости в серверную папку и клиентскую через: npm install</li>
	<li>3. В Терминале открыть папку сервера и написать npm run dev, затем так-же открыть папку клиента и написать npm start</li>

</ul>
